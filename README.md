PCQQ协议的Python语言实现

本项目由纯 Python3 标准库实现，简单易懂无第三方依赖库

支持手机扫码登录与账号密码登录，在终端环境下需要安装 pillow 库打印二维码

通过 Generator 和 asyncio 异步协程实现事件的处理与调度

`session.token` 是登录完成后用于下次直接重连的令牌文件

`cache.db` 是存储群信息以及成员信息的数据库文件(可能更新的不及时)
